# Настройка ИИ помощника

## Как это работает

Приложение использует **Google Gemini 2.5 Flash Lite API** для анализа естественного языка и создания задач.

**Пример использования:**
- Пользователь пишет: "купить молоко завтра"
- ИИ анализирует текст и возвращает структурированные данные
- Приложение автоматически создает задачу

## Установка

### 1. Получить API ключ (бесплатно!)

1. Перейди на https://ai.google.dev/
2. Нажми **"Get API key"** или **"Get free API key"**
3. Выбери **"Create API key in new project"**
4. Скопируй ключ

### 2. Настроить переменные окружения

Отредактируйте файл `config.env` в корне проекта:

```bash
GOOGLE_API_KEY=твой_ключ_здесь
```

Ключ можно получить за 10 секунд, **не требует кредитную карту**!

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

## Использование

Откройте приложение и нажмите кнопку "💬" (чат) внизу экрана.

### Примеры команд

- "Купить молоко"
- "Встреча с боссом в пятницу 14:00"
- "Позвонить маме завтра вечером"
- "Помыть машину в выходной"
- "Купить подарок подруге на день рождения"

## Поддерживаемые категории

- работа
- личное
- покупки
- здоровье
- другое

## Что делает ИИ

1. **Анализирует текст** на естественном языке
2. **Извлекает данные:**
   - Название задачи
   - Описание (опционально)
   - Категория
   - Дата и время (если указаны)
3. **Возвращает JSON** с структурированными данными
4. **Приложение создает задачу** на основе этих данных

## Стоимость

- **АБСОЛЮТНО БЕСПЛАТНО!**
- Лимит: **60 запросов в минуту** (достаточно для мобильного приложения)
- Без кредитной карты, без скрытых платежей

## Troubleshooting

### "GOOGLE_API_KEY не установлен"
Убедитесь что файл `.env` содержит корректный `GOOGLE_API_KEY`

### "Ошибка сети"
Проверьте интернет-соединение и доступ к https://generativelanguage.googleapis.com

### "Ошибка парсинга JSON"
ИИ вернул некорректный JSON. Это может быть редкой ошибкой. Попробуйте снова.

### "Quota exceeded"
Превышен лимит 60 запросов в минуту. Подождите немного и попробуйте снова.
